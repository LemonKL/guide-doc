                                            <h1>动态代理是什么？应用场景？</h1>

<p>动态代理：在运行时，创建目标类，可以调用和扩展目标类的方法。</p>

<p>Java 中实现动态的方式：<a href="https://blog.csdn.net/meism5/article/details/90744045" rel="nofollow" data-token="2072fb3c61f179a67029e74a9131aac5">JDK 中的动态代理</a>&nbsp;和 <a href="https://blog.csdn.net/meism5/article/details/90781518" rel="nofollow" data-token="2d48c03e97ca3a299ccd082b3174d410">Java类库 CGLib</a>。</p>

<p>&nbsp;</p>

<p>应用场景如：</p>

<ul><li>统计每个 api 的请求耗时</li>
	<li>统一的日志输出</li>
	<li>校验被调用的 api 是否已经登录和权限鉴定</li>
	<li>Spring的 AOP 功能模块就是采用动态代理的机制来实现切面编程</li>
</ul><p>&nbsp;</p>
